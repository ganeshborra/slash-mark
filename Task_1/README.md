Task 1 – Task Management Application (Python)
Overview:

This project is a command-line based Task Management Application developed using Python.
It allows users to add, remove, view, and get task recommendations based on priority.

This project is submitted as Task 1 of the internship.

Features:

Add tasks with priority (Low / Medium / High)
Remove tasks by description
View all tasks
Recommend a high-priority task
Stores data persistently using a CSV file

Technologies Used:

Python 3
Pandas
Scikit-learn

Project Files:

main.py – Main application logic

tasks.csv – Stores task data

instructions.txt – Usage instructions

Setup & How to Run:

Install required libraries:

pip install pandas scikit-learn

Run the application:

python main.py
Conclusion

This project demonstrates basic task automation, file handling, and simple use of machine learning in Python as part of an internship task.
